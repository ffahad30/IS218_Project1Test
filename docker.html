Docker

What is Docker?

Docker is an open source containerization platform. It allows you to deploy containers that store the websites and applications you develop. Docker is useful because it allows for an easier facilitation of web and app development due to the containers' efficiency in being deployed from other software such as GitHub, and to other software such as Microsoft Azure. Learn more here: https://docs.docker.com/get-started/overview/.

Docker Installation

  • Install Docker Here: https://docs.docker.com/get-docker/
  • Sign Up for Docker Here: https://hub.docker.com/
  • Access Docker Online Here: https://hub.docker.com/
  • Open the Docker application on your computer

Docker Commands

  • touch Dockerfile - creates Dockerfile file
  • touch docker-compose.yml - creates docker-compose.yml file
  • docker-compose up -d - starts and runs containers
  • docker build -t ( docker username)/(repository name) . - builds docker image
  • docker login azure - logs into azure
  • docker run -dp 80:80 (docker username)/(repository name) - runs image on docker and deploys it on azure
  • docker ps - shows you running containers and their IP addresses
  • docker context show - shows what context(computer) you are using
  • docker context create aci (name of context) - creates aci context
  • docker --context (name of context) run -p 80:80 nginx - runs nginx web server
  • docker stop (name of container) - stop the web server/container
  • docker rm (name of container) - removes the container
  • docker prune - releases resources that have been allocated for container groups
