Web Servers

What is a Web Server?

Web servers are used to deliver web pages to a user's browser. The two kinds of web servers are Apache and Nginx. Apache and Nginx are open source softwares that use a user's directory request to send them the web pages. Apache and Linux are what make websites available to users, and therefore it is essential to know how to use them if you want your website to be accessible. Learn more here: https://kinsta.com/blog/nginx-vs-apache/.

How to Use Apache and Nginx

In the process of building a website, when you get to the step of running the command "docker --context (name of context) run -p 80:80 nginx" to deploy your Docker container on Azure, include "nginx" at the end of the command to use the Nginx web server, or remove "nginx" at the end of the command to use the Apache web server.
