About This Website

This website will teach you how to build an HTML 5 website. With this tutorial, you will learn to build your website using Bootstrap as the CSS framework. You will also learn how to use Git, Vi in Git, Docker, Web Servers (Apache and Nginx), Linux Commands, and Azure. Your website will be created using the following steps:

1. Using the command line, Linux commands, and Vi in Git Bash, create the HTML and CSS files for your website.

2. Push your HTML and CSS files from Git to Docker.

3. Build your image on Docker.

4. Run your image on Docker and deploy your Docker container to Azure.

5. Using Git Bash, find your website's IP address.

6. In a web browser, enter the IP address to access your website.
